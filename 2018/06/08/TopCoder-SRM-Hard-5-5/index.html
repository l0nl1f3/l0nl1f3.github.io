SRM 582 1000 SemiPerfectPower

题意

求 $[L, R]$ 中，有多少数可以被表示为 $ab^c$ 的形式，其中 $a,b,c$ 均为正整数，且$a<b$

题解

首先容易发现 $c=2$ 或 $c=3$, 其他情况都可以用这两种表示

考虑 $n = ab^2$ 的部分，枚举 $a$, 答案就是 $\sum_a [\mu_a \ne 0]\max(\sqrt {\frac n a} - a, 0)$

得到 $a\le \sqrt {\frac n a}$, 即 $a^3\le n$, 暴力枚举即可

同理我们可以得到 $n = ab^3$ 的有 $\sum_{a^4\le n}{} \sqrt[3]{\frac{n}{a}} - a$, 其中 $a$ 没有三次的因子

完结撒花

还要考虑去重，如果 $n = ax^2 = by^3$

$n = by^3 = (by) y^2$, 找到一个极大的 $k^2|by$, 此时 $n = \frac{by}{k^2}(yk)^2$, 若不存在 $n = ax^2$, 必然有 $\frac{by}{k^2} \ge yk$，化简得 $k \le b^{\frac{1}{3}}$

我们希望能在 $n = by^3$ 的部分排除能表示为 $n = ax^2$ 的数，枚举 $b, y, k$, 设 $c(x)=$ x没有三次的质因子,$d(x)=[\mu_x \ne 0]$

考虑给 $\frac{by}{k^2}$ 约去 $(k^2, b)$, 设 $k’ = \frac k g, b’=\frac b g, y’ = \frac{y}{k’}$

由于 $d(ab)=d(a)d(b)[(a,b)=1]$

$\sum_{d|n} \mu(d) = [n = 1]$, 带入

然后就是

暴力枚举 $b, k$, 暴力枚举 $b’$ 的无平方银子，预处理最后一个和式即可

SRM 562 1000 InducedSubgraphs

题意

给一个树 $T$ 和一个整数 $k$, 问有多少种给 $T$ 重标号的方式满足 $T’$ 中任何一个长度为 $k$ 的标号区间对应的导出子图都是联通的
$n\le 41$

题解

一开始看这个题你发现如果 $2k\le n$, 那么最后 $T’$ 的 $1\ldots k$ 肯定是个父亲标号 > 孩子标号的子树，后 $k$ 个点同理，中间只能是长为 $n-2k$ 的链，枚举链端点，链的标号方式只有一种，考虑 $dp$ 两边的，$f(x)$ 表示 $x$ 的子树标号方案数，$x$ 肯定是标号最大的, 给每一个子树 $y$ 分配 $s_y$ 个编号，方案数就是 $f(x) = (s_x-1)!\sum_y \frac{f(y)}{s_y!}$, 这个部分是 $n^4$ 的

若 $2k > n$, $[n-k+1,n]$ 肯定是一个联通块。设这些点为白点，设前 $n-k$ 个为红点，后 $n-k$ 个为蓝点，考虑移动这个长度为 $k$ 的区间，相当于红点和蓝点互换，取一个白点为根(显然最后重心一定是白点), 每个不含白点的子树内部只有一钟颜色，且满足父亲标号 > 孩子标号，设 $f(i,j,k)$ 为 $i$ 子树内有 $j$ 个红点 $b$ 个红点的方案数，转移显然等价于二维 egf 的乘法。数据范围很小，暴力转移即可,最后答案就是 $f[1][n-k][n-k]$，还要再乘上白点随便标号的方案数 $(2n-k)!$, 复杂度看起来是 $O(n^5)$ 的，实际上和转移与树形背包无太大区别，复杂度是 $O(n^4)$ 的

SRM 551 900 SweetFruits

题意

有 n 个点，其中一些点是好的，每个好的点有一个价值

现在要把这 n 个点连成一棵树，定义一个点是非常好 的，当且仅当它本身和它的一个邻居都是好的

对于一棵树，定义它的价值为所有非常好的点的价值和，求有多少价值<= maxValue 的树

n <= 40, maxValue <= 10^9

题解

如果树上的非常好点个数相同，不考虑价值，所有情况都是等价的，于是可以先meet in the middle 出选了 $k$ 个好点，价值 $\le mx$ 的方案数

考虑求 $T_k$ 为恰好 $k$ 个非常好点的方案数，如果我们强制所有 $(i, j)$，其中
$j$ 为假点或 $i, j$ 同时为前 $k$ 个好点中的一个的 $(i, j)$ 连边，求生成树个数，这样能求出有至多 $k$ 个好点的方案数 $g_k$, 易得 $T_k = g_k - \sum_{j<k} {k\choose j} T_j$

总复杂度 $O(n^4+n2^{\frac{n}{2}})$

SRM 550 850 ConversionMachine

题意

给定两个等长的仅有 $(a,b,c)$ 的串 $s, t$, 每次操作可以使一个 $a\to b, b\to c, c\to a$，各有一定代价，问 $s\to t$ 花费不超过 $w$ 的方案数

$|s|\le 11$

题解

首先可以很轻易的得到步数上限 $m$, 然后不超过这个上限xjb操作都是不会超过 $w$ 的，就抛掉了一层限制，

设 $f[i][j][k][l]$ 为用了 $i$ 步，剩 $j$ 个还差 1 步的字符, $k$ 个差 $2$ 步， $l$ 个差 $0$ 步的方案数，其中 $i+j+k=n$, 矩乘优化转移，总复杂度 $|s|^6\log w$

如果 Hard 全是这个难度的我一天可以做 10 道 (括号删)

SRM 548 1000 KingdomAndCities

题意

求有多少个 $n$ 个点 $k$ 条边，无重边无自环的无向联通图个数，且满足前 $m$ 个点度数恰好为 $2$

$n\le 50, m\le 2, k\le 50$

题解

不管这个 $m$ 个 $2$ 度点的情况可以 $f(i,j)$ 表示 $i$ 个点 $j$ 条边的答案，枚举 $i+1$ 所在联通块大小容斥即可，复杂度 $n^2k^2$

然后讨论一下，$m=1$ 的情况就是删掉 $1$ 号点后剩 $1$ 或 $2$ 个联通块，枚举一个联通块大小算算就好了。
$m = 2$ 的其实也是这样做一下 。。 就是讨论起来头顶有点凉飕飕的

原来二度点可以视为插在边中间，然后随便讨论一下和原来被拆的边之间的连边情况，就会得到一个比这个简单多的式子 。。。羞耻地删掉这个傻逼代码。。。
